CareBot

CareBot is a health monitoring and prediction platform that allows users to track essential health metrics such as BMI, glucose, insulin, and more. It performs real-time risk-level predictions and health assessments while maintaining secure user sessions using Express-MySQL integration.

Tech Stack: Node.js, Express.js, MySQL, Express-MySQL-Session, TypeScript, Postman, Git.
